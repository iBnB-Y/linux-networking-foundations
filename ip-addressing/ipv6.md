
# IPv6 (Internet Protocol Version 6)

## What is IPv6?
IPv6 is the newer version of IP designed to replace IPv4 due to address exhaustion.

## Format
IPv6 uses 128 bits and is written in hexadecimal:
Example:
2001:db8::1

## Key Features
 Very large address space
 No need for NAT
 Uses multicast instead of broadcast
 Better routing efficiency

## Why IPv6 Matters
Modern networks, cloud systems, and mobile networks increasingly rely on IPv6.
