

## How DNS Works
1. User enters domain name
2. Resolver checks cache
3. Query goes to DNS servers
4. IP address is returned

## DNS Record Types
 A – IPv4 address
 AAAA – IPv6 address
 CNAME – Alias
 MX – Mail server
 NS – Name server
 TXT – Verification & security

## Ports
 UDP 53
 TCP 53
